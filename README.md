# Entity Framework Core Events and Diagnostic Listeners

<img src="https://www.yogihosting.com/wp-content/uploads/2025/11/EFCore-Events-Diagnostic-Listeners.png" alt="Entity Framework Core Events and Diagnostic Listeners"  title="Entity Framework Core Events and Diagnostic Listeners">

Entity Framework Core Events and Diagnostic Listeners are very helpful in debugging our EF Core code. Events are called when something happens in EF Core codes.Diagnostic listeners allow listening for any EF Core event for obtaining diagnostic information of the app. In this repository we are going to implement each of these 2. Tutorial <a href="https://www.yogihosting.com/events-diagnostic-listeners-entity-framework-core/" target="_blank">link</a>

## Version
.NET 10.0

## Instruction
To create database change the connection string in the `appsettings.json` file and then perform migration - `PM> add-migration Migration1` `PM> Update-Database`

## What is covered
The following things are implemented in this repository:

- [x] Implementation of Entity Framework Core Events
- [x] Implementation of Diagnostic Listeners
